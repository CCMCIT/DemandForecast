    What each command does
    Check setup
    Checks the database connection, .env, and watch folder. Read-only.
    Ingest a file
    Loads one CSV file into the database staging tables.
    Ingest a folder
    Loads every CSV in a folder. Files already loaded are skipped.
    Process next N files
    Turns the next N ingested files into voyages (oldest first).
    Process pending
    Turns all ingested files into voyages. Files whose type has no processor are skipped.
    Gate-activity pending
    Turns all ingested gate-activity files into gate-activity records.
    Import status
    Shows how many files loaded successfully vs failed. Read-only.
